工具說明 — 工作流程
[上傳文件] → [設定選項] → [分析] → [確認摘要] → [輸出文件] ↓ ↓ ↓ ↓ ↓ PDF/Word 預算/來源 Claude AI 可手動修正 5種格式下載
① 上傳邀請函
支援 PDF、Word (.docx)、TXT、EML。可多選上傳。若 AI 讀取有限，請將文字貼入④。
② 補充資訊
勾選年度預算狀態，選擇活動來源（用於預算申請信）。
③ 分析
AI 讀取文件後，將重要資訊填入右側摘要表格。需先在 Settings 設定 API Key。
④ 額外提示
摘要有遺漏或錯誤，輸入修正說明後重新按「分析文件」。
⑤ 輸出文件
活動協議草稿 → .docx  ｜  法務審批表格 → .docx
預算申請信草稿 → .md  ｜  回覆/追蹤信 → .md
活動摘要報告 → 開啟列印對話框 → 選「儲存為 PDF」
⚙ Settings
API Key、Base URL（預設 Anthropic，可改其他相容 API）、Model 選擇。設定儲存於瀏覽器 localStorage。
⚙ Settings — API 設定
⚠ API Key 儲存於瀏覽器 localStorage，僅限本機使用。
請勿在公用電腦儲存，亦勿將此 HTML 連同 Key 外傳。
